redis

  1. apt-get install redis-server
  2. 安装完成后，Redis服务器会自动启动，我们检查Redis服务器程序,执行ps -aux|grep redis
  3. 通过启动命令检查Redis服务器状态,执行：netstat -apn|grep 6379

  4. 安装Redis服务器，会自动地一起安装Redis命令行客户端程序。在本机输入redis-cli命令就可以启动，客户端程序访问Redis服务器。执行：redis-cli

    redis-server //启动服务
    sudo pkill redis-server //关闭Redis服务
    sudo netstat -tunpl | grep 6379
    pstree -p | grep redis
    redis-cli

    sudo service redis-server restart
    redis-server
    redis-cli

mongod 备份任务

  1. 新建任务sh
    mkidr tasks
vim blog.backup.sh

#!/bin/sh

backUpFolder=/home/kingdee/backup/
datenow=`date +%Y%m%d%H%M`
backFileName=blog_$date_now

cd $backUpFolder
mkdir -p $backFileName

mongodump -h 127.0.0.1:27017 -d kd -o $backFileName

tar zcvf $backFileName.tar.gz $backFileName

rm -rf $backFileName

  1. 执行定时备份任务

    crontab -e
    04 10 * sh /home/kingdee/tasks/blog.backup.sh //任务执行时间
